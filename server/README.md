

* [Node.js](https://nodejs.org/en/download/)
* [MongoDB](https://www.mongodb.com)
* **npm** This comes with Node.js, but make sure you check if you have it anyway

### Installing packages

Install packages

```
npm i
```

### Running the app

To run the app (dev. mode)

```
run mongo in cmd using mongod
then,

npm start
```
